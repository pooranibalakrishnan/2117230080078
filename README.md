# Affordmed Campus Hiring Evaluation - Stage 1

## Priority Notification Inbox System

This project implements a notification prioritization system using Node.js and JavaScript.

The application fetches notifications, calculates priority scores based on type and recency, and displays the top important notifications.

---

# Features

- Fetch notifications from API
- Calculate priority score
- Sort notifications dynamically
- Display top notifications
- Handle API failure gracefully
- Modular code structure

---

# Technologies Used

- JavaScript
- Node.js
- Axios

---

# Run Project

```bash
node src/index.js
```
